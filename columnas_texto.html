COLUMNAS DE TEXTO

Estructura y Etiquetas

  • column-count definie el número de columnas en las que queremos dividir el texto.
  • column-gap definine el espacio en píxeles que existirá como mínimo entre las columnas.
  • column-rule define a la línea separadora que hay entre las columnas y contiene 3 valores ('grosor de la línea', 'el tipo de separador' y 'su color').

column-count

column-count:2;

column-gap

column-gap:10px;

column-rule

column-rule:4px dotted gray;

Escogemos el texto al que asignaremos en columnas

Este texto puede ser un párrafo muy largo o bien una lista de palabras.

< div id="columnas">
    < li> Amazonas: Leticia. < /li>
    < li> Antioquia: Medellín. < /li>
    < li> Atlántico: Barranquilla. < /li>
    < li> Bolívar: Cartagena de Indias. < /li>
    < li> Boyacá: Tunja. < /li>
    < li> Caldas: Manizales. < /li>
    < li> Caquetá: Florencia. < /li>
    < li> Casanare: Yopal. < /li>
    < li> Cauca: Popayán. < /li>
< /div>

Ingresamos las características de las columnas en CSS

Seguidamente el código que indicará el número de columnas, la separación y el tipo de separadores irán en el espacio reservado para el código CSS.

#columnas{
    column-count:4;
    column-gap: 20px;
    column-rule:4px double rgb(0, 70, 110);
}
