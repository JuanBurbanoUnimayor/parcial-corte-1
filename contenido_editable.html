CONTENIDO EDITABLE

Para que un contenido sea editable (es decir, para que el usuario pueda modificar, eliminar o añadir texto) desde el propio navegador, dicho texto tiene que estar ubicado dentro de un

o de una etiqueta semántica (article, section...) con la propiedad 'contenteditable=true':

Codigo para un contenido editable

< section id="editable" contenteditable="true">
      < h2>Escribe 3 o más ciudades de Colombia< /h2>
        < ul>
          < li>Popayán< /li>
          < li>Cali< /li>
          < li>Medellin< /li>
        < /ul>
< /section>

Ejemplo Contenido Editable

Escribe 3 o más ciudades de Colombia

  1. Popayán
  2. Cali
  3. Medellin
