FORMULARIOS

Etiquetas para formularios

Lo primero a tener en cuenta hace referencia a la estructura general. Así, todas las etiquetas que vamos a utilizar para el formulario tienen que estar ubicadas dentro de las etiquetas del formulario < form > y < /form >.

Estructura y Etiquetas

  • type indica qué tipo de datos estamos pidiendo.
  • size especifica el tamaño del campo requerido.
  • title incluye un texto adicional cuando el cursor esté encima.
  • name es la forma de identificar al valor cuando se envie la información.
  • aotofocus señala el campo con el que el formulario iniciará por defecto, además solo puede haber uno en todo el formulario.
  • required indica que el campo es obligatorio y si no se rellena no se podrá enviar la información.

Usuario - (text)

< input type="text" size="40" title="Escribe tu usuario. " name="usuario" autofocus required/ >

Contraseña - (password)

< input type="password" size="7" title="Escribe la contraseña" name="password" required/ >

Correo Electrónico - (email)

< input type="email" placeholder="usuario@dominio.com" size="40" title="Escribe tu correo electrónico" name="mail" required/ >

Fecha - (date)

< input type="date" value="2000-01-01" name="nacimiento" title="Ingresa tu fecha de nacimiento"/ >

Rangos - (range)

< input type="range" min="1" max="7" value="2" step="1" title="Calificanos" name="calificacion"/ >

Datos numericos - (number)

< input type="number" min="0" max="10" step="1" title="Cuantos hermanos tienes" name="hermanos"/ >
